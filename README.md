# IROS-2026
Supplementary Material: Non-Local Inference for Occluded DLO Manipulation via Tactile-Conditioned Diffusion Policies

This repository contains supplementary video demonstrations for the IROS 2026 submission detailing our closed-loop framework for robotic "wire fishing". The videos showcase the performance of the proposed tactile-conditioned diffusion policy in generating multi-modal recovery behaviors for occluded deformable linear object (DLO) manipulation.

IROS 2026 L-shape pipe success example.mp4
Description: Demonstrates the system successfully navigating the base training environment, which consists of a single 90-degree bend.

IROS 2026 U-shape and T-shape pipe success example compressed.mp4
Description: Exhibits preliminary zero-shot generalization in novel, complex environments.

IROS 2026 U-shape and T-shape pipe failed example compressed.mp4
Description: Provides a transparent look at the limits of tactile-only non-local inference in highly complex exploratory environments.
